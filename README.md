
## Generals

Sometimes Google Chrome's bookmark sync messes up my bookmarks. The old version of the bookmark comes out on the unused notebook and is mixed in. what a mess.


## Usage


### Pre-requisites
   - pip3 install beautifulsoup4
### Steps
    1. In bookmark editor of chrome, put high-priority links/folders at the front of the bookmark bar
    2. export bookmark from chrome
    3. python app.py
    4. import new bookmark to chrome
    simple.
